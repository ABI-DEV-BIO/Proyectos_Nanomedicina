# Nanomedicina Computacional 🧬

## 📌 Descripción
Este repositorio documenta mis desarrollos en la intersección entre la nanorobótica y la medicina. El objetivo es modelar comportamientos de agentes microscópicos y automatizar procesos lógicos aplicados a la salud.
